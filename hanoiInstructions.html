Game #2: Tower Instructions

Estimated 3 minutes.
Requires a mouse.
The goal of this game is to move all discs from the first peg to the last peg such that the discs are stacked with the largest disc on the bottom to smallest disc on top, all while adhering to the following rules:

- One disc must be moved at a time
- A larger disc must not be placed on a smaller disc
- Only the top disc can be moved from a stack

The game will end after the task has been completed.
